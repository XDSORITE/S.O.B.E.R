2) Create a new notebook or open a exisitng on
3) On the top bar there is a menu for run click it and select "Kaggle Jupyter Server"
4) A menu on the right side will open click start session and you will reciece vscode compatible url, copy it
5) How to add in vscode https://code.visualstudio.com/docs/datascience/jupyter-notebooks#_connect-to-a-remote-jupyter-server
6) Then run each cell 1 after the other starting the top
